# 📱 Progressa

O **Progressa** é um aplicativo mobile focado em produtividade, organização de estudos e criação de hábitos saudáveis.

---

## 🚀 Objetivo

Ajudar estudantes a:

- 📅 Organizar seus estudos
- 📊 Acompanhar progresso por matéria
- 🎯 Definir metas semanais
- 💧 Criar hábitos saudáveis (como hidratação e leitura)

Acreditamos que **grandes resultados vêm de pequenas evoluções diárias**.

---

## 🎨 Design

O layout do app foi criado no **Figma**, com foco em:

- Interface moderna
- Usabilidade simples
- Experiência visual limpa

---

## 🛠️ Tecnologias utilizadas

- Kotlin
- Android Studio
- XML (Layouts)
- Git & GitHub

---

## 📱 Funcionalidades

### ✅ Implementadas
